readUInt64LE function

int readUInt64LE (ByteData bytes, int offset)

Implementation

int readUInt64LE (ByteData bytes, int offset) {
  final a = bytes.getUint32(offset, Endian.little);
  var b = bytes.getUint32(offset + 4, Endian.little);
  b *= 0x100000000;
  isUint(b + a, 64);
  return b + a;
}
